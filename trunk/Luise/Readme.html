Luise Readme

Thanks for downloading Luise! Here's what's included in this package:
  • Luise.exe - The main executable, just double-click.
Requirements
  • Visual Basic 6.0 SP5 Runtime (if you're running Windows 2000 or later then you almost certainly have this already, only install if you get an error message that it's missing).
  • Phantasy Star III rom image, do not ask me for a copy.

30-Second Tutorial

  1. Start or continue a game in Gens or Genecyst (or another emulator that creates .GS# save state files)
  2. Save the game state - this does not edit battery save games, only save state files
  3. Run Luise
  4. Browse to the save state file (rom-name.GS#)
  5. Edit stuff
  6. Click the save button
  7. Repeat as desired
